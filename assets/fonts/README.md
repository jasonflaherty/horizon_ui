# Fonts

Phase 1 themes use Material / platform typefaces with distinctive metrics
(letter-spacing, weight, tabular figures). Optional branded families can be
registered here and wired through `HorizonTypographyTokens` later:

- Luxury Coastal — editorial serif (e.g. Literata)
- Cyber Surf — geometric sans / mono (e.g. Space Grotesk, JetBrains Mono)
